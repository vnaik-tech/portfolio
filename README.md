## üöÄ Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/jayramgodkar/portfolio.git
   cd portfolio
   ```

2. Open `index.html` in your browser or use a local server:
   ```bash
   python -m http.server 8000
   # or
   npx serve
   ```

3. Visit `http://localhost:8000` in your browser

## üìù Customization

### Update Personal Information
- Replace email, phone, and location in the contact section
- Update project descriptions and links
- Add your own projects and skills

### Modify Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --color-primary: #34656D;
    --color-secondary: #334443;
    --color-accent: #FAEAB1;
    --color-light: #FAF8F1;
}
```

### Add Social Media Links
Update the social links in the contact section to point to your profiles:
```html
<a href="https://github.com/vnaik-tech" target="_blank" class="social-btn">
    <i class="fab fa-github"></i>
</a>
```
